What is this about anyway?

Long story short, it is just a little hobby of mine. I've always been interested in classic web design, vintage computing and graphical design. So I decided to fuse all three together all whilst using legacy software to build it. I hope you like it.

Any long term goals?

As far as right now goes, this remains a hobby and a non-serious project, something I can relax whilst building and potentially add some cool features whilst learning a bit of HTML code in the process. And maybe connect with some new folk who also have a similar interest if they make it here.

Software and Trivia:

I built this website using Macromedia Dreamweaver MX 2004 and Adobe Fireworks CS6. The design philosophy is loosely based around 1990s operating systems and GUI design such as Silicon Graphics IRIX, Microsoft Windows and Amiga Workbench.
